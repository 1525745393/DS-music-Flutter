# DS Music Flutter

一款基于 Flutter 开发的跨平台音乐应用。

## 功能特性

- 跨平台支持（iOS / Android / Web / Desktop）
- 现代化 UI 设计
- 本地音乐播放与管理

## 技术栈

- Flutter 3.x
- Dart

## 快速开始

### 环境要求

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0

### 安装依赖

```bash
flutter pub get
```

### 运行项目

```bash
flutter run
```

## 项目结构

```
lib/
├── main.dart          # 应用入口
├── screens/           # 页面
├── widgets/           # 通用组件
├── models/            # 数据模型
├── services/          # 业务服务
└── utils/             # 工具函数
```
