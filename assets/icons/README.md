# 图标资源

## 应用图标
请将以下尺寸的 PNG 图标放到本目录：

| 密度 | 尺寸 | 用途 |
|------|------|------|
| mdpi | 48x48 | 普通 |
| hdpi | 72x72 | 高 |
| xhdpi | 96x96 | 更高 |
| xxhdpi | 144x144 | 超高 |
| xxxhdpi | 192x192 | 极高 |

并在 `android/app/src/main/res/` 对应 mipmap-* 目录中放置。

## 启动图
- `LaunchImage.png` - iOS 启动图
- Android 启动：`android/app/src/main/res/drawable/launch_background.xml`

## 状态/通知图标
- `mipmap/ic_launcher.png` - 通知中心小图标（建议白底透明）
